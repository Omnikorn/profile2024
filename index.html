This is a fun little web application that allows the user to select a country and through various API calls scours the internet to present a collection of relevant information about the selected country. When loaded the application uses the user current location to present data on the users current country. The data includes general information about the country including capital city and flag in addition to a weather forecast, current news, Wikipedia articles and a currency conversion widget that converts the countries currency to US dollars. .
